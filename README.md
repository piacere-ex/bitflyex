# Bitflyex

[Bitflyex](https://hex.pm/packages/bitflyex) is a bitFlyer Lightning API libraries in Elixir. Here is an example:

```elixir
iex> 

```

See the [online documentation](https://hexdocs.pm/bitflyex).

## Installation

Add to your ```mix.exs``` file:

```elixir
def deps do
  [
    { :bitflyex, "~> 0.0.1" }
  ]
end
```
